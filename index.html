Projects

Lorem

Lorem is a tool which can be used to manage events. Events can be created, managed and deleted by administrators. Normal users can sign up for events and rate them upon completion. The administrator's panel is written in Java and the user's page is written in C#. You can view a demo by clicking on the video, please note that it is in Dutch, but there are English subtitles.

Cllct

Cllct is a bookmarking tool which you can use to collect links to websites. Links can be sorted into categories and given a name and a color. It is also possible to search for links and there is a dark mode. I created this tool because I was unsatisfied with the bookmark management options in Chrome. The bookmarks would often fall out of sync with each other on different computers and it was not accessable from a computer where sync was not turned on (such as a friends computer). It is written in C#.
